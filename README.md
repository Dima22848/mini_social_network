# Mini Social Network

Full Stack social network built with NestJS and Next.js.

## Technologies

- NestJS
- Next.js
- PostgreSQL
- Prisma
- Redis
- Socket.IO
- Docker
- BullMQ

## Features

- JWT Authentication
- Access & Refresh Tokens
- Session Management
- Friends System
- Followers & Following
- Private Chats
- Group Chats
- Online Presence
- Typing Indicators
- Posts
- Comments
- Reactions
- File Uploads
- Notifications
